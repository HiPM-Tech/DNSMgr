## 项目架构分析

### 整体架构
- **前后端分离架构**：前端使用 React 19 + TypeScript + Tailwind CSS，后端使用 Node.js + Express + TypeScript
- **模块化设计**：清晰的目录结构，前端按功能模块组织，后端按路由、服务、中间件等分层
- **多数据库支持**：支持 SQLite、MySQL、PostgreSQL 三种数据库
- **DNS 提供商集成**：支持阿里云、Cloudflare、DNSPod 等多种 DNS 服务提供商

### 技术栈
- **前端**：React 19.2.4、TypeScript 5.9.3、Tailwind CSS 3.4.17、React Query 5.80.7、React Router 7.2.0
- **后端**：Node.js、Express 4.18.2、TypeScript 5.3.3、Swagger 文档
- **数据库**：SQLite (默认)、MySQL、PostgreSQL
- **认证**：JWT、2FA (TOTP)、WebAuthn、OAuth
- **安全**：CORS、Rate Limiting、Content Security Policy
